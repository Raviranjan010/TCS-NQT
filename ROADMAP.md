# TCS NQT — Repository Roadmap

## Why This Order?

The recommended study order is: **Coding → Quant → Reasoning → Verbal → Email/Paragraph Writing**

**Reasoning behind this order:**
1. **Coding** has the highest per-question impact for Digital/Prime band and takes the longest to improve from zero. Start here when your mind is fresh.
2. **Quantitative Ability** is the most concept-heavy section and the biggest score differentiator in Foundation. Formulas and tricks need time to sink in.
3. **Reasoning** is highly pattern-driven. Once you learn the patterns, improvement is rapid — suited to mid-preparation.
4. **Verbal** benefits most from consistent daily exposure (reading). Techniques can be learned quickly; vocabulary grows slowly. It fits well near the end with daily reading kept up throughout.
5. **Email / Paragraph Writing** takes the least preparation time and should be done last — templates can be internalized in 2–3 sessions.

```
Phase 1: Coding (Arrays → Strings → HashMap → Math → Sorting → Searching → Recursion → DP → Stack → LL → Matrix)
   ↓
Phase 2: Quant (DI → Number System → TSD → Time & Work → Percentage → P&L → SI/CI → Ratio → Avg → P&C → Probability)
   ↓
Phase 3: Reasoning (Series → Coding-Decoding → Seating → Puzzle → Syllogism → Blood → Analogy → Direction → ...)
   ↓
Phase 4: Verbal (RC → Fill Blanks → Error Detection → Para Jumbles → Synonyms → ...)
   ↓
Phase 5: Email Writing (templates + timed practice)
   ↓
Phase 6: Mock Tests → Review → Last Day Revision
```
