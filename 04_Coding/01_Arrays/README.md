# Arrays

Arrays are the most fundamental data structure and the single highest-frequency topic in the TCS NQT coding section.
Mastering the 7 core patterns (Kadane's, Prefix Sum, Sliding Window, Two Pointer, Frequency Array, Rotate, Merge)
covers approximately 80% of all Easy–Medium array problems you will encounter.

- **TCS NQT Frequency**: 🔴 High — almost guaranteed in every coding slot
- **Difficulty**: Easy to Medium (occasionally Hard for Prime)
- **Expected problems per slot**: 1–2 out of 2 total problems will involve arrays
