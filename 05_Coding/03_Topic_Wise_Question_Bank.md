# Topic-Wise Question Bank

Solve these in C++ only. First try without looking at solutions, then compare with [04_CPP_Solutions_With_Explanation.md](04_CPP_Solutions_With_Explanation.md).

## A. Numbers And Basic Logic

### Q1. Sum Of Digits

Input an integer `n`. Print the sum of its digits.

Example:

```text
Input: 4729
Output: 22
```

### Q2. Palindrome Number

Input an integer `n`. Print `YES` if it is a palindrome, otherwise print `NO`.

### Q3. Prime Number

Input an integer `n`. Print `YES` if it is prime, otherwise print `NO`.

### Q4. GCD And LCM

Input two integers `a` and `b`. Print their GCD and LCM.

### Q5. Armstrong Number

Input an integer `n`. Print `YES` if it is an Armstrong number, otherwise print `NO`.

## B. Arrays

### Q6. Second Largest Element

Input `n` and an array. Print the second largest distinct element. If it does not exist, print `-1`.

### Q7. Reverse Array

Input `n` and an array. Print the array in reverse order.

### Q8. Rotate Array Left By K

Input `n`, array, and `k`. Rotate the array left by `k` positions and print it.

### Q9. Missing Number From 1 To N

Input `n - 1` numbers containing values from `1` to `n` with one missing number. Print the missing number.

### Q10. Maximum Subarray Sum

Input `n` and an array that may contain negative numbers. Print the maximum possible sum of a non-empty contiguous subarray.

## C. Strings

### Q11. Palindrome String

Input a string. Print `YES` if it is a palindrome, otherwise print `NO`.

### Q12. Count Vowels And Consonants

Input a string. Count vowels and consonants. Ignore spaces and non-alphabet characters.

### Q13. First Non-Repeating Character

Input a lowercase string. Print the first non-repeating character. If none exists, print `-1`.

### Q14. Check Anagram

Input two lowercase strings. Print `YES` if they are anagrams, otherwise print `NO`.

### Q15. Reverse Words In A Sentence

Input a sentence. Reverse the order of words.

Example:

```text
Input: tcs nqt coding
Output: coding nqt tcs
```

## D. Sorting And Hashing

### Q16. Sort 0s, 1s And 2s

Input an array containing only `0`, `1`, and `2`. Sort it in one pass.

### Q17. Frequency Of Each Element

Input an array. Print each distinct element and its frequency in increasing order of element.

### Q18. Majority Element

Input an array. Print the element occurring more than `n/2` times. If none exists, print `-1`.

### Q19. Kth Largest Element

Input `n`, array, and `k`. Print the kth largest element.

### Q20. Pair Sum Exists

Input `n`, array, and target. Print `YES` if any pair sums to target, otherwise print `NO`.

## E. Prefix Sum, Sliding Window, Two Pointers

### Q21. Equilibrium Index

Input an array. Print an index where left sum equals right sum. If none exists, print `-1`.

### Q22. Maximum Sum Subarray Of Size K

Input `n`, array, and `k`. Print the maximum sum among all subarrays of size `k`.

### Q23. Longest Substring Without Repeating Characters

Input a string. Print the length of the longest substring with all unique characters.

### Q24. Remove Duplicates From Sorted Array

Input a sorted array. Print the array after removing duplicates.

### Q25. Move Zeroes To End

Input an array. Move all zeroes to the end while keeping the order of non-zero elements.

## F. Matrix

### Q26. Diagonal Sum

Input an `n x n` matrix. Print the sum of primary and secondary diagonals. Do not double count the center element.

### Q27. Transpose Matrix

Input a matrix and print its transpose.

### Q28. Spiral Traversal

Input a matrix and print elements in spiral order.

## G. Greedy And Stack

### Q29. Minimum Coins

Given Indian denominations `{2000, 500, 200, 100, 50, 20, 10, 5, 2, 1}`, input an amount and print the minimum number of coins/notes.

### Q30. Balanced Parentheses

Input a string containing brackets `()[]{}`. Print `YES` if brackets are balanced, otherwise print `NO`.

### Q31. Stock Buy And Sell Once

Input daily prices. Print the maximum profit possible with one buy and one sell.

### Q32. Activity Selection

Input start and finish time of activities. Print the maximum number of non-overlapping activities.

## H. TCS-Style Mixed Problems

### Q33. Product Except Self

Input an array. Print an array where each position contains product of all other elements. Do not use division.

### Q34. Compress String

Input a string. Replace consecutive repeated characters with character followed by count.

Example:

```text
Input: aaabbc
Output: a3b2c1
```

### Q35. Leaders In Array

An element is a leader if every element to its right is smaller. Print all leaders from left to right.

### Q36. Subarray Sum Equals K

Input an array and integer `k`. Print the number of subarrays whose sum equals `k`.
