## Daily Rule

Every day solve:

- 2 easy problems
- 1 medium problem
- 1 dry run by hand
- 1 previous wrong problem

## Mock Test Format

Set timer for 90 minutes:

1. Easy problem: 25 to 30 minutes
2. Medium problem: 45 to 55 minutes
3. Final testing: 5 to 10 minutes

## How To Review Mistakes

For each wrong problem, write:

```text
Problem:
Wrong reason:
Correct pattern:
Edge case missed:
Final code idea:
Redo date:
```

This turns mistakes into revision material.
