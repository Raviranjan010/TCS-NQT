# Study Plan — Flexible Schedule (1 Day to 2 Weeks)

## How to Use This Plan

Find your available time in the table below and follow that row's schedule. Every plan ends at the same place (exam-ready) — shorter plans sacrifice depth but preserve coverage of highest-frequency topics.

**Before starting any plan, do this in 15 minutes:**
1. Read [Exam_Pattern.md](./00_Getting_Started/Exam_Pattern.md) — know the structure
2. Read [Marks_Distribution.md](./00_Getting_Started/Marks_Distribution.md) — know the priorities

## Daily Study Habits (For All Plans)

| Habit | Why It Matters |
|-------|---------------|
| Always time yourself | Untimed practice builds the wrong muscle — you need speed |
| Write down every mistake | Review your mistake list before the next session |
| Don't study the same topic two sessions in a row | Spaced practice solidifies memory better than massing |
| Cap each session at 2.5 hours | Cognitive fatigue kills quality; take breaks |
| Sleep 7–8 hours | Memory consolidation happens during sleep — non-negotiable |
| Do not study new content the night before the exam | Revision only — new content causes anxiety, not improvement |
