# Time Speed Distance - Shortcuts & Speed Hacks

## ⚡ 1. Constant Distance Ratio Rule
*   **Shortcut:** If distance is constant, speed is inversely proportional to time ($S_1/S_2 = T_2/T_1$).
*   **Real Number Demo:** If a person increases speed from 40 km/hr to 50 km/hr (ratio 4:5), the time ratio is 5:4.
