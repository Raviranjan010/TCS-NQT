# Time Speed Distance - Flashcards

*   **Card 1: What is relative speed in same direction?**
    *   *Answer:* The difference of the two speeds ($S_1 - S_2$).
*   **Card 2: How to convert km/h to m/s?**
    *   *Answer:* Multiply the speed by $\frac{5}{18}$.
*   **Card 3: How does constant distance affect speed and time ratios?**
    *   *Answer:* Speed is inversely proportional to time ($S_1 : S_2 = T_2 : T_1$).
*   **Card 4: Define upstream and downstream.**
    *   *Answer:* Downstream is rowing along the stream direction; upstream is rowing against the stream direction.
