# Time Speed Distance - Cheatsheet

*   **Speed conversion:** $1\text{ km/hr} = \frac{5}{18}\text{ m/s}$, $1\text{ m/s} = \frac{18}{5}\text{ km/hr}$
*   **Relative Speed:**
    *   Opposite Direction: $S_{rel} = S_1 + S_2$
    *   Same Direction: $S_{rel} = |S_1 - S_2|$
*   **Average Speed:**
    *   Equal Distances: $S_{avg} = \frac{2xy}{x+y}$
    *   Equal Times: $S_{avg} = \frac{x+y}{2}$
*   **Boats & Streams:**
    *   Downstream: $D = U+V$, Upstream: $U_{up} = U-V$
    *   Boat Speed: $U = (D+U_{up})/2$, Stream Speed: $V = (D-U_{up})/2$
