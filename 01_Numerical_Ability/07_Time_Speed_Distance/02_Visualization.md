                   +-------------------+-------------------+

            Same Direction                          Opposite Direction

### Comparison Table:
| Parameter | Same Direction | Opposite Direction |
| :--- | :--- | :--- |

| **Meeting Time** | $T = \text{Separation} / (S_1 - S_2)$ | $T = \text{Separation} / (S_1 + S_2)$ |
