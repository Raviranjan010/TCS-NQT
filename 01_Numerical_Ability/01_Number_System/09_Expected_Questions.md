# 01 Number System - Expected Questions

This file contains 15 TCS NQT-style expected questions based on 2022-2025 patterns.

---

**Q1.** When a positive integer N is divided by 8, it leaves a remainder of 7. Find the remainder when N^2 + 2N + 3 is divided by 8.
*   **Type:** Divisibility and Remainders
*   **Answer:** By remainder theorem, replace N with 7 in N^2 + 2N + 3. Remainder = (7^2 + 2*7 + 3) mod 8 = 2
*   **Time target:** 45 seconds
*   **TCS frequency:** High (appears in 70% of papers)
