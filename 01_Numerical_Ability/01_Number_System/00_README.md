# 01_Number_System Directory Overview

Welcome to the **Number System** module of the TCS NQT study guide. This directory is designed to build foundational speed and accuracy for divisibility, factors, remainders, and unit digit calculations.

---

## 📊 Module Weightage & Exam Relevance
*   **Section**: Section 1A (Numerical Ability) & Section 1C (Advanced Quantitative)
*   **Average Questions**: 3–4 Questions
*   **Target Time**: Under 75 seconds per question
*   **Prerequisites**: Arithmetic operations, division tables.
