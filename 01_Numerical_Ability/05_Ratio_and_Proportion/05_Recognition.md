# Ratio and Proportion - Pattern Recognition Guide

## 1. The Coin Bag Pattern
*   **Trigger Word:** *"A bag contains ₹1, 50p, and 25p coins in ratio X:Y:Z, and the total value is ₹V."*
*   **Strategy:** Convert ratios to value equivalents by multiplying by denomination weight.

### Immediate Worked Example:
A bag contains ₹1, 50p, and 25p coins in the ratio $2:3:4$. If the total value is ₹180, find the number of 50p coins.
*   **Value conversion:** Let coins be $2x, 3x, 4x$.
    $$2x(1) + 3x(0.50) + 4x(0.25) = 180 \implies 2x + 1.5x + x = 180 \implies 4.5x = 180 \implies x = 40$$
*   **50p coins:** $3x = 3 \times 40 = 120$ coins.
