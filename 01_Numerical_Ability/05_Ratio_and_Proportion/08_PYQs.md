# Ratio and Proportion - Past Year Questions (PYQs)

### PYQ 1. Proportional constants
*   **Question:** What number must be added to each of 6, 15, 20, and 43 so that the resulting numbers are in proportion?
*   **Pattern ID:** RATIO_PROP_ADD_PYQ
*   **Approach:** Set up equation: $\frac{6+x}{15+x} = \frac{20+x}{43+x}$ and solve, or test option values.
*   **Solution:**
    $$(6+x)(43+x) = (15+x)(20+x) \implies 258 + 49x + x^2 = 300 + 35x + x^2$$
    $$14x = 42 \implies x = 3$$
*   **Shortcut:** Try small values from options. For $x=3$: $\frac{9}{18} = \frac{23}{46} = \frac{1}{2}$. Correct.
*   **Variation & Trap:** Be careful with sign changes (adding vs subtracting).
