# 05 Ratio and Proportion - Expected Questions

This file contains 15 TCS NQT-style expected questions based on 2022-2025 patterns.

**Q1.** If A : B = 3 : 4 and B : C = 4 : 5, find the compound ratio A : B : C and evaluate the value of A if the total sum of A, B, and C is 1200.
*   **Type:** Compound Ratios and Allocations
*   **Answer:** Ratio is 3:4:5. Sum of terms = 12. A's share = (3 / 12) * 1200 = 300.
*   **Time target:** 35 seconds
*   **TCS frequency:** High (appears in 70% of papers)
