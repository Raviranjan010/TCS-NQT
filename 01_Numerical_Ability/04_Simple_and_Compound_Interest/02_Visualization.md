# Simple and Compound Interest - Visual Guide & Models

```
                      Interest Calculation Flow
                                 |
           +---------------------+---------------------+
           |                                           |
      Simple Interest (SI)                    Compound Interest (CI)
    (Constant base: Principal)             (Shifting base: Principal + Interest)
           |                                           |
    Interest is constant                      Interest grows exponentially
    Formula: SI = P*R*T/100                   Formula: A = P*(1 + R/100)^T
```

### Comparison Table:
| Parameter | Simple Interest (SI) | Compound Interest (CI) |
| :--- | :--- | :--- |
| **Base** | Original Principal only | Principal + Accrued Interest |
| **Yearly Interest** | Constant each year | Increases every year |
| **Growth Type** | Linear ($I \propto T$) | Exponential ($A \propto (1+R)^T$) |
