# Probability - Practice Advanced

**Q1.** An item is produced by three machines A, B, and C. Machine A produces $30\%$ of the items, B produces $50\%$, and C produces $20\%$. The defect rates are $3\%$, $2\%$, and $4\%$ respectively. If a randomly selected item is found to be defective, find the probability it was produced by machine B.
> 🎯 Hint: Apply Bayes' Theorem: $P(B | D) = \frac{P(B)P(D|B)}{P(A)P(D|A) + P(B)P(D|B) + P(C)P(D|C)}$.
<details>
<summary>✅ Solution</summary>
*   **Approach:** Bayes' Theorem.
*   **Step-by-step:**
    *   $P(A) = 0.30, P(B) = 0.50, P(C) = 0.20$
    *   $P(D|A) = 0.03, P(D|B) = 0.02, P(D|C) = 0.04$
    *   Numerator $= 0.50 \times 0.02 = 0.010$.
    *   Denominator $= (0.3 \times 0.03) + (0.5 \times 0.02) + (0.2 \times 0.04) = 0.009 + 0.010 + 0.008 = 0.027$.
    *   Probability $= 0.010 / 0.027 = 10/27$.
*   **Answer:** 10/27
*   **Difficulty:** ⭐⭐⭐⭐⭐ | **Target Time:** 90s
</details>
