# Permutation & Combination - FAQ

**Q1. What is the difference between permutation and combination?**
*   **Answer:** Permutation is about arranging items where order is critical. Combination is about selecting items where order is irrelevant.

**Q2. How does the addition rule differ from the multiplication rule?**
*   **Answer:** Use addition rule for mutually exclusive choices (A OR B). Use multiplication rule for sequential operations (A AND B).

**Q3. What is circular permutation?**
*   **Answer:** The number of unique ways to arrange items in a ring structure, where rotations of the same relative order are considered identical.

**Q4. How to solve word arrangement problems?**
*   **Answer:** Count total letters, find repeat frequencies, treat restricted groups (like "vowels together") as a single block, arrange the blocks, and then perform internal block arrangements.
