# Averages - Shortcuts & Speed Hacks

## ⚡ 1. Deviation Method for Averages
*   **Shortcut:** Assume a baseline average, calculate the deviation of each number, average the deviations, and add to baseline.
*   **Real Number Demo:** Find the average of $98, 102, 95, 105, 100$.
    *   Assume baseline $= 100$.
    *   Deviations: $-2, +2, -5, +5, 0 \implies \text{Sum} = 0$.
    *   Average $= 100 + 0 = 100$.
