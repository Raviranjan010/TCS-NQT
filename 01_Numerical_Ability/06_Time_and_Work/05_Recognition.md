# Time and Work - Pattern Recognition Guide

## 1. Alternate Days Work Pattern
*   **Trigger Phrasing:** *"A and B work on alternate days, starting with A. Find the total time to complete the work."*
*   **Strategy:** Find the work done in a 2-day cycle, then multiply to get close to total work, and handle the remaining work step-by-step.

### Immediate Worked Example:
A takes 10 days, B takes 20 days. They work on alternate days starting with A.
*   **LCM Setup:** Work $= 20$ units. Efficiency: A $= 2$, B $= 1$.
*   **2-Day Cycle:** Work done in 2 days $= 2 + 1 = 3$ units.
*   **Multiplication:** $6$ cycles $\times 2\text{ days} = 12\text{ days}$ doing $6 \times 3 = 18$ units.
*   **Remaining:** $20 - 18 = 2$ units left.
*   **Day 13:** A works and does 2 units. Work finished. Total $= 13$ days.
