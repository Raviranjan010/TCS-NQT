# Time and Work - Flashcards

*   **Card 1: If efficiency is doubled, what happens to time?**
    *   *Answer:* Time is halved.
*   **Card 2: How do we represent emptying pipes in calculation?**
    *   *Answer:* As negative efficiency.
*   **Card 3: What is the significance of "Work = 1" in fractional method?**
    *   *Answer:* It represents 100% of the single task completion.
*   **Card 4: How does alternate days work affect total duration?**
    *   *Answer:* It requires grouping work into 2-day cycles and solving the remainder separately.
