# Time and Work - Shortcuts & Speed Hacks

## ⚡ 1. Efficiency-Ratio Method
*   **Shortcut:** If A is $k$ times as efficient as B, then the ratio of days taken is:
    $$\text{Ratio of Days (A:B)} = 1 : k$$
*   **Real Number Demo:** A is twice as efficient as B. If B takes 20 days, A takes $20 / 2 = 10$ days.
