# Percentage Shortcuts & Speed Tricks

## 1. Commutative Percentages ($A\%$ of $B = B\%$ of $A$)
$$\text{Formula: } A\% \text{ of } B = B\% \text{ of } A$$

### Why this works:
$$A\% \text{ of } B = \frac{A}{100} \times B = \frac{A \times B}{100} = \frac{B}{100} \times A = B\% \text{ of } A$$

### Demonstration:
Calculate $18\%$ of $50$ mentally.

*   **Standard Method**: $\frac{18}{100} \times 50 = 0.18 \times 50 = 9$ (requires paper calculations).
*   **Shortcut Method**: Swap variables to find $50\%$ of $18$.
    $$50\% \text{ of } 18 = \frac{18}{2} = 9\text{ (done mentally in 2 seconds)}$$

## 2. The Splitting Method for Mental Percentages
Break down complex percentages into sum/difference of easy baseline values ($10\%$, $5\%$, $1\%$).

```
                      Find 56% of 400
                             |
             +---------------+---------------+
             |                               |
          Find 50%                        Find 5% (1/10 of 50%)
          (400 / 2)                       (200 / 10 = 20)
             |                               |
             +---------------+---------------+
                             |
                          Find 1% (400 / 100 = 4)
                             |
                       Sum: 200 + 20 + 4 = 224
```

### Demonstration:
Find $56\%$ of $400$.

*   **Standard Method**: $\frac{56}{100} \times 400 = 56 \times 4 = 224$.
*   **Shortcut Method**:
    *   $50\%$ of $400 = 200$
    *   $5\%$ of $400 = 20$ (which is $50\% \div 10$)
    *   $1\%$ of $400 = 4$
    *   Total $= 200 + 20 + 4 = 224$.

## 3. Product Constancy Scale (Fraction Jumps)
If a product $P = A \times B$ is constant, any percentage change in $A$ forces a balanced change in $B$ on a fixed fractional scale:

$$\text{Increase in } A = \frac{1}{x} \implies \text{Decrease in } B = \frac{1}{x+1}$$
$$\text{Decrease in } A = \frac{1}{x} \implies \text{Increase in } B = \frac{1}{x-1}$$

### Demonstration:
If a worker's hours are reduced by $20\%$, by how much must their productivity increase to keep output constant?

*   **Standard Method**: Let original hours $= 100$, output $= 100$. New hours $= 80$. New productivity $= \frac{100}{80} = 1.25 \implies 25\%$ increase.
*   **Shortcut Method**:
    *   Identify hours decrease: $20\% = \frac{1}{5}$. Here, $x = 5$.
    *   Apply the jump: Since it's a decrease, the increase needed is:
        $$\frac{1}{x-1} = \frac{1}{5-1} = \frac{1}{4} = 25\%\text{ increase}$$

## 4. Successive Change Quick Approximation
When two successive changes $x\%$ and $y\%$ are small (both under $5\%$), you can ignore the compound term $\frac{xy}{100}$.

$$\text{Net Change} \approx x + y$$

### Demonstration:
A town's population increases by $3\%$ in Year 1 and $2\%$ in Year 2. Estimate the net increase.

*   **Standard Method**:
    $$\text{Net Change} = 3 + 2 + \frac{3 \times 2}{100} = 5 + 0.06 = 5.06\%$$
*   **Shortcut Method**:
    $$\text{Estimate} = 3 + 2 = 5\%$$
    (The error is just $0.06\%$, which is negligible for eliminating wrong options in MCQs instantly).
