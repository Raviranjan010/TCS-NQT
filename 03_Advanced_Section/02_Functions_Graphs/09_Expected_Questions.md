**Q1.** If f(x) = x^2 - 4x + 3 and g(x) = 2x - 1, find the value of the composite function f(g(3)).
*   **Type:** Composite Functions
*   **Answer:** g(3) = 2*3 - 1 = 5. f(g(3)) = f(5) = 5^2 - 4*5 + 3 = 25 - 20 + 3 = 8.
*   **Time target:** 45 seconds
*   **TCS frequency:** Medium (appears in 45% of papers)
