This file contains 15 TCS NQT-style expected questions based on 2022-2025 patterns.

**Q1.** Evaluate the value of x: log_2(x) + log_2(x-2) = 3.
*   **Type:** Logarithmic Equations
*   **Answer:** log_2(x(x-2)) = 3 -> x(x-2) = 2^3 = 8 -> x^2 - 2x - 8 = 0 -> (x-4)(x+2) = 0. Since log argument must be positive, x = 4.
*   **Time target:** 50 seconds
*   **TCS frequency:** Medium (appears in 50% of papers)
