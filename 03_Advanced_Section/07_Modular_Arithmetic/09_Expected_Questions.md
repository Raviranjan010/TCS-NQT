# 07 Modular Arithmetic - Expected Questions

This file contains 15 TCS NQT-style expected questions based on 2022-2025 patterns.

**Q1.** Find the value of x such that 3x = 4 (mod 7).
*   **Type:** Modular Linear Congruences
*   **Answer:** Multiply both sides by modular inverse of 3 mod 7 (which is 5 because 3*5 = 15 = 1 mod 7). x = 4 * 5 = 20 mod 7 = 6.
*   **Time target:** 45 seconds
*   **TCS frequency:** Medium (appears in 40% of papers)
