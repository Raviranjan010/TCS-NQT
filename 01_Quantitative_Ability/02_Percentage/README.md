# Percentage - Module Overview

This module covers the core concepts of Percentages, which form the mathematical foundation for several Quantitative Ability topics (including Profit & Loss, SI/CI, and Data Interpretation) in the TCS NQT exam.

---

## Metadata

*   **TCS NQT Frequency:** `🔴 High` (Appears in almost every exam cycle with 2-3 direct questions)
*   **Difficulty Level:** `🟢 Easy` to `🟡 Medium`
*   **Core Pre-requisites:** Basic Arithmetic, Ratio Conversions

---

## Study Guide & Syllabus

1.  **Fundamental Calculations:** Understanding the base value and percentage multipliers.
2.  **Fractional Equivalents:** Converting percentages to fractions (e.g., $16.67\% = 1/6$) to save computation time.
3.  **Successive Percentage Changes:** Successive increases and decreases (compounding).
4.  **Product Constancy Rule:** Balancing changes to keep the product constant (e.g., Price $\times$ Consumption = Expenditure).
5.  **Percentage Points (pp) vs. Percentage Change (%):** Interpreting growth graphs.
