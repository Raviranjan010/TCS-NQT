# Time, Speed & Distance — Tricks & Traps

## 1. The Average Speed Trap
**Trap:** Adding two speeds and dividing by 2 to get average speed.
**Reality:** If distances are equal, use harmonic mean: 2S1S2/(S1+S2)
**Example:** 60 km/h to office, 40 km/h back. Average ≠ 50. It = 2(60)(40)/(60+40) = 48 km/h.

## 2. Train Length Forgotten
**Trap:** Using only platform length when calculating crossing time.
**Reality:** Time = (Train Length + Platform Length) / Speed — BOTH lengths matter.

## 3. Unit Confusion in Trains
**Trap:** Speed in km/h and length in meters — forgetting to convert.
**Fix:** Always convert to same units before substituting.
- If speed in km/h, convert meters to km OR convert km/h to m/s.

## 4. Opposite-Direction Start Direction
**Trap:** Confusing which formula applies when trains go in same vs opposite directions.
**Memory aid:** "Opposite = add speeds (they meet faster), Same = subtract speeds (one catches other slowly)"

## 5. Boats — Don't Add Stream Speed to Upstream
**Trap:** Upstream speed = B + S (wrong)
**Fix:** Upstream = B - S (stream opposes the boat)

## 6. Relative Speed vs Closing Speed
**Trap:** Using absolute speed instead of relative speed for chasing problems.
**Fix:** If A chases B, relative speed = S_A - S_B. Time to close gap h = h/(S_A - S_B)

## 7. TCS Option Elimination Trick
For "find average speed" problems, the answer is always LESS than the arithmetic mean:
- Eliminate any option ≥ (S1+S2)/2
- Narrow down instantly to 1-2 options
