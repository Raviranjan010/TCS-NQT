# TSD — Cheatsheet

**S = D/T | D = ST | T = D/S**

| Conversion | Value |
|------------|-------|
| km/h → m/s | × 5/18 |
| m/s → km/h | × 18/5 |

| Scenario | Formula |
|----------|---------|
| Relative (opposite) | S1+S2 |
| Relative (same dir) | \|S1-S2\| |
| Train-Pole | L/S |
| Train-Platform | (L+P)/S |
| Two Trains | (L1+L2)/(S1±S2) |
| Downstream | B+S |
| Upstream | B-S |
| Still Water | (D+U)/2 |
| Stream | (D-U)/2 |
| Avg Speed (equal dist) | 2S1S2/(S1+S2) |
| Avg Speed (equal time) | (S1+S2)/2 |
