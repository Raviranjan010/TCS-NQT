# Time, Speed & Distance — Formulas

## Core Triangle
| Find | Formula |
|------|---------|
| Speed | D / T |
| Distance | S × T |
| Time | D / S |

## Unit Conversion
- 1 km/h = 5/18 m/s
- 1 m/s = 18/5 km/h

## Relative Speed
| Scenario | Relative Speed |
|----------|---------------|
| Same direction | \|S1 - S2\| |
| Opposite direction | S1 + S2 |

## Train Problems
| Scenario | Formula |
|----------|---------|
| Cross a pole | L_train / S |
| Cross a platform | (L_train + L_platform) / S |
| Two trains, same dir | (L1 + L2) / (S1 - S2) |
| Two trains, opposite | (L1 + L2) / (S1 + S2) |

## Boats & Streams
$$D_s = B + S \quad U_s = B - S$$
$$B = \frac{D_s + U_s}{2} \quad S = \frac{D_s - U_s}{2}$$

**Derivation:** Downstream, stream aids the boat; upstream it opposes. Adding and subtracting eliminates the stream component.

## Average Speed (Equal Distances)
$$\text{Avg Speed} = \frac{2 S_1 S_2}{S_1 + S_2}$$

**Derivation:** If distance = d each way:
- Time1 = d/S1, Time2 = d/S2
- Avg Speed = 2d / (d/S1 + d/S2) = 2S1S2/(S1+S2)

## Meeting Problems
- Two people start simultaneously from opposite ends, distance D:
  $$\text{Meeting Time} = \frac{D}{S_1 + S_2}$$
- One person chases another, head start = h:
  $$\text{Catch Time} = \frac{h}{S_1 - S_2}$$
