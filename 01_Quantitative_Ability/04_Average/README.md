# Average - Module Overview

This module covers the mathematical principles of averages, weighted averages, average speeds (and the harmonic mean speed trap), and group change-of-average problems using the deviation method.

---

## Metadata

*   **TCS NQT Frequency:** `🔴 High` (Appears as 2 direct word problems per exam cycle)
*   **Difficulty Level:** `🟢 Easy` to `🟡 Medium`
*   **Core Pre-requisites:** Addition, Division, and Ratio scaling

---

## Syllabus Coverage

1.  **Arithmetic Mean:** Definition, sum of elements, and properties of consecutive number series.
2.  **Weighted Average:** Averages of multiple groups of unequal sizes.
3.  **Average Speed:** Total distance / total time. Equal distance (harmonic mean) vs. equal time (arithmetic mean).
4.  **Deviation Method:** Rapidly calculating changes in averages when elements are added, removed, or replaced.
