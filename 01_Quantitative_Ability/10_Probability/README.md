# Probability — Module Overview

**TCS NQT Frequency:** 🔴 High (1-2 questions per exam)
**Difficulty:** 🟡 Medium
**Prerequisites:** Permutation & Combination, Fractions

## Syllabus (TCS NQT Level)
1. Basic Probability: P(E) = Favorable / Total
2. Complementary Events: P(E') = 1 - P(E)
3. Addition Rule: P(A∪B)
4. Multiplication Rule: P(A∩B) for independent events
5. Dice problems, Card problems, Coin problems
