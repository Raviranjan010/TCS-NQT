# Permutation & Combination — Cheatsheet

| Concept | Formula |
|---------|---------|
| Permutation | nPr = n!/(n-r)! |
| Combination | nCr = n!/r!(n-r)! |
| Circular seating | (n-1)! |
| Necklace | (n-1)!/2 |
| Identical elements | n!/p!q!... |
| Vowels together | (n-k+1)! × k! |
| At least 1 | 2^n - 1 (for subsets) |
| Handshakes | nC2 = n(n-1)/2 |
| Triangles | nC3 |
| Total subsets | 2^n |

## P vs C Decision
- Arrange/Order/Rank/Password → **P**
- Select/Choose/Committee/Team → **C**
