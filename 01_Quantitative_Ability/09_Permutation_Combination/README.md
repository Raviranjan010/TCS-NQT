# Permutation & Combination — Module Overview

**TCS NQT Frequency:** 🔴 High (1-2 direct questions per exam)
**Difficulty:** 🟡 Medium
**Prerequisites:** Factorials, Basic Counting

## Syllabus (TCS NQT Level — not CAT level)
1. Fundamental Counting Principle
2. Permutation: nPr formula — ordered arrangements
3. Combination: nCr formula — unordered selections
4. Distinguishing when to use P vs C
5. Circular Permutation (basics only)
6. Simple word arrangement problems
