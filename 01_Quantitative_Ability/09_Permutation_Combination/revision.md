# Permutation & Combination — Quick Revision

## Decide: P or C?
- Order matters → **nPr = n!/(n-r)!**
- Order doesn't matter → **nCr = n!/r!(n-r)!**

## Key Values
| Formula | Result |
|---------|--------|
| nCr = nC(n-r) | Symmetry |
| nC0 = nCn = 1 | Boundaries |
| nC1 = n | Single selections |
| n people handshakes | nC2 = n(n-1)/2 |
| Triangles from n pts | nC3 |
| Circular seating | (n-1)! |
| Total subsets | 2^n |
| Repeated letters | n!/p!q!... |

## Vowels Together
(n-k+1)! × k!  where k = vowel count
