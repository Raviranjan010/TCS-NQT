# Profit & Loss - Module Overview

This module covers the core principles of Profit, Loss, Discount, Markup, and Dishonest Dealer (False-Weight) problems. These concepts are among the most frequently tested arithmetic topics in the TCS NQT exam.

---

## Metadata

*   **TCS NQT Frequency:** `🔴 High` (Appears as 2-3 direct questions per exam cycle)
*   **Difficulty Level:** `🟢 Easy` to `🟡 Medium`
*   **Core Pre-requisites:** Percentages, Ratios, and Basic Fractions

---

## Study Guide & Syllabus

1.  **Fundamental Relations:** Cost Price (CP), Selling Price (SP), and Marked Price (MP) connections.
2.  **Markup & Discount:** Formulas for Markup %, Discount %, and SP calculations based on MP.
3.  **Successive Discounts:** Calculating single equivalent discount from multiple successive discounts.
4.  **Dishonest Dealers:** Solving false-weight problems using ratio balancing.
