# Time & Work — Exam Shortcuts & Hacks

These shortcuts are designed to bypass long fractions and multi-step equations in time-and-work problems.

---

## 1. LCM Shortcut (Efficiency Calculation)

Instead of using fraction additions like $rac{1}{12} + rac{1}{15} + rac{1}{20}$:

1.  **Assume Total Work:** LCM of denominators. For 12, 15, and 20, LCM is 60 units.
2.  **Calculate Individual Efficiencies:**
    *   $E_1 = 60 / 12 = 5$
    *   $E_2 = 60 / 15 = 4$
    *   $E_3 = 60 / 20 = 3$
3.  **Combine Efficiencies:**
    *   $E_{	ext{total}} = 5 + 4 + 3 = 12$ units/day.
4.  **Find Time Taken:**
    *   Time $= 60 / 12 = 5$ days.
*   *Time saved:* No common denominator fraction math.

## 3. Efficiency Ratios to Time Ratios

*   Since Time and Efficiency are inversely proportional, if the efficiency ratio of A and B is $x:y$, the ratio of time taken by them to complete the same work is:
    $$	ext{Time Ratio} = y : x$$

---

## 4. Alternate Days Work Routine

If A and B work on alternate days starting with A:
1.  Calculate 2-day combined work block $= E_A + E_B$.
2.  Find the largest multiple of this block that is $\le$ total work.
3.  Find remaining work and allocate to A (and B if still remaining).\n
