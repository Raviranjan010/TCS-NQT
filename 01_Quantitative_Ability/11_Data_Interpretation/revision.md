# Data Interpretation — Revision Strategy

## 5-Step DI Approach

1. **Read Title** — understand what data measures
2. **Check Units** — thousands? millions? plain numbers?
3. **Scan All Data** — don't calculate yet
4. **Read Each Question** — identify what needs to be found
5. **Calculate Precisely** — use correct formula

---

## Core DI Formulas

| Question Type | Formula |
|--------------|---------|
| % Change | (New-Old)/Old × 100 |
| % of Total | Part/Total × 100 |
| Value from % | (P/100) × Total |
| Ratio | A/B (simplify) |
| Average | Sum / Count |

---

## DI Chart Types

| Chart | Key Feature | Common Questions |
|-------|------------|-----------------|
| Table | Rows × Columns | Totals, ratios, differences |
| Bar Graph | Height = value | Max, min, % change |
| Pie Chart | Sector = % | Values, ratios |
| Line Graph | Trend | Highest growth, fall |

---

## Quick Tips

- For approximation: if options differ by >5%, estimate freely
- For exact: compute carefully, especially with 2-3 decimal choices
- Always double-check which year/category the question refers to
