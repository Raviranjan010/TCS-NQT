# Data Interpretation — Cheatsheet

## Formulas
| Calculation | Formula |
|-------------|---------|
| % Change | (New-Old)/Old × 100 |
| % of Total | Part/Total × 100 |
| Value from % | Pct/100 × Total |
| Average | Sum/Count |
| Ratio | Simplify A/B |
| Pie angle | Pct/100 × 360 |

## DI Reading Checklist
- [ ] Read chart title
- [ ] Check axis units
- [ ] Understand row/column headers
- [ ] Identify data range (years, categories)
- [ ] Note any "total" row/column

## Common Traps
- Old value = base for % change (not new)
- "By how much" = subtraction (not %)
- Scale units: "in thousands" means ×1000
- Pie % ≠ degrees

## Approximation Guide
- Options differ by >5%? Estimate.
- Options close? Compute precisely.
