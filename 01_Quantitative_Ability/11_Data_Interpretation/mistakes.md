# Data Interpretation — Common Mistakes

## 1. Wrong Year as Base for % Change
- **Mistake:** % change from 2022 to 2023 = (2022-2023)/2023 × 100
- **Fix:** ALWAYS use the earlier year (OLD value) as denominator

## 2. Misreading Chart Scale
- **Mistake:** Bar height = 12, reading as 12 when axis says "in hundreds" = 1200
- **Fix:** Always multiply by the scale factor mentioned

## 3. Including Wrong Rows in Average
- **Mistake:** "Average of 3 departments" — counting 4 by mistake
- **Fix:** Count items explicitly before summing

## 4. Confusing % and Absolute Values
- **Mistake:** "Which has 30% more?" — subtracting percentages from pie chart
- **Fix:** Convert percentages to actual values first, then compare

## 5. Pie Chart Degree to Percentage
- **Mistake:** Reading a 90° sector as 90%
- **Fix:** 90° = 90/360 × 100 = 25%

## 6. Ratio Simplification Error
- **Mistake:** 240:160 = 3:2 (correct) vs 24:16 = 3:2 (same, but candidates sometimes skip simplification and match wrong option)
- **Fix:** Always simplify ratio to lowest terms

## 7. "By how much" vs "By what %"
- "By how much" = absolute difference (subtraction)
- "By what %" = percentage difference (divide by base, multiply by 100)
