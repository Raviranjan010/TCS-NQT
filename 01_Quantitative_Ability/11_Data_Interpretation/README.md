# Data Interpretation — Module Overview

**TCS NQT Frequency:** 🔴 High (2-4 questions per set, 1-2 sets per exam)
**Difficulty:** 🟡 Medium
**Prerequisites:** Percentage, Average, Ratios, Basic Arithmetic

## What is Data Interpretation?
DI questions present data in a visual format (table, bar graph, pie chart, line graph) followed by 4-5 questions that test your ability to read, extract, and compute answers from that data.

## Syllabus Coverage
1. Table-based DI — reading rows/columns accurately
2. Bar Graph DI — comparing values across categories
3. Pie Chart DI — sector percentages and totals
4. Line Graph DI — trends and rate of change
5. Combined DI — two data sources (table + graph)

## TCS NQT Pattern
- Data is presented once, followed by 4-5 questions
- Questions test: exact values, percentages, ratios, averages, differences
- No graph paper — all values are exact or rounded
