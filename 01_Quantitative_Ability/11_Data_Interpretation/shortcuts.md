# Data Interpretation — Shortcuts

## 1. Approximate Before Computing
For % change questions: estimate if the answer options are ≥ 5% apart.
- "Sales went from 240 to 300": change = 60/240 = 25% (easy: 1/4)
- Don't compute (300-240)/240 × 100 longhand if you recognize 60/240 = 1/4

## 2. Pie Chart — Find Value Without Calculator
If pie shows 35% of 1,40,000:
- 35% = 30% + 5%
- 30% of 1,40,000 = 42,000; 5% = 7,000 → Total = 49,000

## 3. Ratio to Percentage
- A:B = 3:2 means A = 3/5 = 60%, B = 2/5 = 40%
- Useful when finding individual values from ratios

## 4. Table Reading Speed
For row-column tables:
1. Read ALL column headers first
2. Read the specific row question refers to
3. Pick exactly the cell asked — don't read adjacent cells

## 5. Growth Comparison Without Full Calculation
To find which year has highest % growth:
- Look for year where (Increase / Previous Year Value) is highest
- Small increase on small base can beat large increase on large base

## 6. "By how much" = Subtraction
"By how much did X exceed Y?" = X - Y (no percentage)
"By what % did X exceed Y?" = (X-Y)/Y × 100

## 7. Average of Two Values Short Method
Average of a and b = a + (b-a)/2 = (a+b)/2
For 3+ values: always add all then divide.
