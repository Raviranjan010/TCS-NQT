# Data Interpretation — Formulas

## 1. Percentage Change
$$\% \text{ Change} = \frac{\text{New Value} - \text{Old Value}}{\text{Old Value}} \times 100$$

## 2. Percentage of Total
$$\% = \frac{\text{Part}}{\text{Total}} \times 100$$

## 3. Value from Percentage (Pie Chart)
$$\text{Value} = \frac{\text{Percentage}}{100} \times \text{Total}$$

## 4. Ratio from Two Values
$$\text{Ratio} = \frac{A}{B} \text{ (simplify using GCD)}$$

## 5. Average from Data Set
$$\text{Average} = \frac{\text{Sum of all values}}{\text{Count of values}}$$

## 6. Growth Rate
$$\text{Growth Rate} = \frac{\text{Current} - \text{Previous}}{\text{Previous}} \times 100$$

## 7. Pie Chart Central Angle
$$\text{Central Angle} = \frac{\text{Percentage}}{100} \times 360°$$
