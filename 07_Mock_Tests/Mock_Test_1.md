## Section 4 — Coding (1 Problem | Suggested Time: 20 Min)

### Q31 — Equilibrium Index of an Array

#### Problem Description
Write a program to find the **equilibrium index** of a given 0-indexed integer array `nums`. 

An equilibrium index is an index `i` such that the sum of elements at lower indices is equal to the sum of elements at higher indices. More formally, `sum(nums[0] ... nums[i-1]) == sum(nums[i+1] ... nums[n-1])`. 
*   If `i == 0`, the left-side sum is assumed to be `0`.
*   If `i == n - 1`, the right-side sum is assumed to be `0`.

Return the **leftmost equilibrium index**. If no such index exists, return `-1`.

#### Input Format
*   First line contains an integer `N`, the size of the array.
*   Second line contains `N` space-separated integers representing the elements of the array.

#### Output Format
*   Print a single integer representing the leftmost equilibrium index.

#### Constraints
*   $1 \le N \le 10^5$
*   $-10^4 \le nums[i] \le 10^4$

#### Sample Input 1
```text
6
1 7 3 6 5 6
```

#### Sample Output 1
```text
3
```
*Explanation: The equilibrium index is 3 because: `nums[0] + nums[1] + nums[2] == 1 + 7 + 3 = 11`, and `nums[4] + nums[5] == 5 + 6 = 11`.*

#### Sample Input 2
```text
3
1 2 3
```

#### Sample Output 2
```text
-1
```
*Explanation: There is no equilibrium index.*
