### Q1 — HCF / LCM
*   **Correct Answer:** C (24)
*   **Derivation:**
    1.  The required number must divide $148, 246, 623$ leaving remainders $4, 6, 11$.
    2.  This means the number must perfectly divide:
        *   $148 - 4 = 144$
        *   $246 - 6 = 240$
        *   $623 - 11 = 612$
    3.  Therefore, the required number is the Highest Common Factor (HCF) of $144, 240, 612$.
    4.  Find the prime factorizations:
        *   $144 = 2^4 \times 3^2$

        *   $612 = 2^2 \times 3^2 \times 17^1$

### Q5 — Compound Interest
*   **Correct Answer:** A (15 years)
*   **Derivation:**

    2.  Under compound interest, the principal grows geometrically.
    3.  If it doubles in 5 years ($P \rightarrow 2P$ in 5 years), it will double again in the next 5 years ($2P \rightarrow 4P$ in 10 years overall).
    4.  It will double once more in the following 5 years ($4P \rightarrow 8P$ in 15 years overall).
    5.  Thus, it takes $3 \times 5 = 15$ years to become 8 times.

### Q10 — Algebra
*   **Correct Answer:** A (2 or 0.5)

## Section 4 — Coding Solution (Q31)

### Approach (Two-Pointer Technique - O(N) Time, O(1) Space)
1.  We maintain a pointer `insertPos` at the beginning of the array.
2.  Iterate through the array with a pointer `i`. If `nums[i]` is non-zero, we swap it with `nums[insertPos]` and increment `insertPos`.
3.  This moves all non-zero elements to the front while preserving their order, and naturally shifts all zeros to the end.

### Optimized C++ Implementation
```cpp
#include <iostream>
#include <vector>

void moveZeroes(std::vector<int>& nums) {
    int insertPos = 0;
    for (int i = 0; i < nums.size(); ++i) {
        if (nums[i] != 0) {
            std::swap(nums[insertPos], nums[i]);
            insertPos++;
        }
    }
}

int main() {
    int n;
    if (!(std::cin >> n)) return 0;
    
    std::vector<int>& nums = *(new std::vector<int>(n));
    for (int i = 0; i < n; ++i) {
        std::cin >> nums[i];
    }
    
    moveZeroes(nums);
    
    for (int i = 0; i < n; ++i) {
        std::cout << nums[i] << (i == n - 1 ? "" : " ");
    }
    std::cout << std::endl;
    return 0;
}
```
