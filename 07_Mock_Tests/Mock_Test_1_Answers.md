## Section 1 — Numerical Ability Solutions

### Q1 — Remainder Theorem
*   **Correct Answer:** A (1)
*   **Derivation:** 
    1.  We need to find $17^{200} \bmod 18$.
    2.  Express 17 in terms of modulo 18: $17 \equiv -1 \pmod{18}$.
    3.  Substitute back into the expression: $17^{200} \equiv (-1)^{200} \pmod{18}$.
    4.  Since 200 is an even number, $(-1)^{200} = 1$.
    5.  Therefore, the remainder is **1**.

### Q2 — Percentage Change
*   **Correct Answer:** C (4% decrease)
*   **Derivation:**
    1.  Apply the successive percentage change formula: $x + y + \frac{xy}{100}$.
    2.  Here, $x = +20$ (increase) and $y = -20$ (decrease).
    3.  Net Change = $20 - 20 + \frac{20 \times (-20)}{100} = 0 - \frac{400}{100} = -4\%$.
    4.  The negative sign indicates a **4% decrease**.

### Q3 — Profit & Loss
*   **Correct Answer:** B (17%)
*   **Derivation:**
    1.  Let the Cost Price (CP) be $100$.
    2.  Marked Price (MP) is $30\%$ above CP $\rightarrow \text{MP} = 100 \times 1.30 = 130$.
    3.  Discount of $10\%$ is allowed on MP $\rightarrow \text{Discount} = 130 \times 0.10 = 13$.
    4.  Selling Price (SP) = $\text{MP} - \text{Discount} = 130 - 13 = 117$.
    5.  Profit = $\text{SP} - \text{CP} = 117 - 100 = 17$ (which is $17\%$ of CP).

### Q4 — Averages
*   **Correct Answer:** C (85 kg)
*   **Derivation:**
    1.  Let the average weight of 8 persons be $W$. Total weight = $8W$.
    2.  When a person weighing 65 kg leaves and a new person (weight $X$) joins, the average increases by $2.5$ kg.
    3.  New total weight = $8(W + 2.5) = 8W + 20$.
    4.  We know: $\text{New Total} = \text{Old Total} - \text{Removed Weight} + \text{Added Weight}$.
    5.  $8W + 20 = 8W - 65 + X \rightarrow 20 = -65 + X \rightarrow X = 85$ kg.

### Q5 — Time & Work
*   **Correct Answer:** C (8.75 days)
*   **Derivation:**
    1.  Let total work be $\text{LCM}(12, 15) = 60$ units.
    2.  Efficiency of A = $60 / 12 = 5$ units/day.
    3.  Efficiency of B = $60 / 15 = 4$ units/day.
    4.  Combined efficiency of A + B = $5 + 4 = 9$ units/day.
    5.  Work completed in the first 3 days together = $9 \text{ units/day} \times 3 \text{ days} = 27$ units.
    6.  Remaining work = $60 - 27 = 33$ units.
    7.  B completes the remaining work alone: $\text{Days} = \text{Remaining Work} / \text{Efficiency of B} = 33 / 4 = 8.75$ days.

### Q6 — Time, Speed & Distance
*   **Correct Answer:** A (300 meters)
*   **Derivation:**
    1.  Convert train speed from km/h to m/s: $72 \text{ km/h} \times \frac{5}{18} = 20$ m/s.
    2.  When a train crosses a pole, the distance covered is equal to the length of the train.
    3.  $\text{Distance} = \text{Speed} \times \text{Time} = 20 \text{ m/s} \times 15 \text{ seconds} = 300$ meters.

### Q7 — Ratio & Proportion
*   **Correct Answer:** B (33)
*   **Derivation:**
    1.  Let the two numbers be $3x$ and $5x$.
    2.  According to the problem: $\frac{3x - 9}{5x - 9} = \frac{12}{23}$.
    3.  Cross-multiply: $23(3x - 9) = 12(5x - 9) \rightarrow 69x - 207 = 60x - 108$.

### Q8 — Probability
*   **Correct Answer:** A (5/33)
*   **Derivation:**
    1.  Total balls = $5 \text{ red} + 7 \text{ blue} = 12$ balls.
    2.  Number of ways to draw 2 balls from 12: $\binom{12}{2} = \frac{12 \times 11}{2} = 66$ ways.
    3.  Number of ways to draw 2 red balls from 5: $\binom{5}{2} = \frac{5 \times 4}{2} = 10$ ways.
    4.  $\text{Probability} = \frac{\text{Favorable outcomes}}{\text{Total outcomes}} = \frac{10}{66} = \frac{5}{33}$.

## Section 4 — Coding Solution (Q31)

### Approach (Prefix Sum Technique - O(N) Time, O(1) Space)
1.  Compute the total sum of all elements in the array `totalSum`.
2.  Iterate through the array while maintaining the sum of elements to the left `leftSum`.
3.  For each index `i`, the sum of elements to the right `rightSum` is simply `totalSum - leftSum - nums[i]`.
4.  If `leftSum == rightSum`, we have found the leftmost equilibrium index. Return `i`.
5.  If the loop completes without finding an equilibrium index, return `-1`.

### Optimized C++ Implementation
```cpp
#include <iostream>
#include <vector>
#include <numeric>

int findEquilibriumIndex(const std::vector<int>& nums) {
    long long totalSum = 0;
    for (int num : nums) {
        totalSum += num;
    }
    
    long long leftSum = 0;
    for (int i = 0; i < nums.size(); ++i) {
        long long rightSum = totalSum - leftSum - nums[i];
        if (leftSum == rightSum) {
            return i; // Leftmost equilibrium index
        }
        leftSum += nums[i];
    }
    
    return -1;
}

int main() {
    int n;
    if (!(std::cin >> n)) return 0;
    
    std::vector<int> nums(n);
    for (int i = 0; i < n; ++i) {
        std::cin >> nums[i];
    }
    
    std::cout << findEquilibriumIndex(nums) << std::endl;
    return 0;
}
```
