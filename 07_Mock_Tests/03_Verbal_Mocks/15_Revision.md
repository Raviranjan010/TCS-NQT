# Verbal Ability - Revision Guide

This guide contains the essential rules and templates for Verbal Ability.

---

## 1. Sentence Correction Rules

### Subject-Verb Agreement
*   **Rule**: The verb must agree with the true subject, not with adjacent nouns or phrases (such as parenthetical clauses like "along with", "as well as").
*   **Real Number Demo**: *"The captain, along with his crew members, **was** (not were) praised."*

### Pronoun Case
*   **Rule**: Use subject pronouns (I, he, she, they) as subjects, and object pronouns (me, him, her, them) as objects.
*   **Real Number Demo**: *"Between you and **me** (not I), the plan has flaws."*

### Dangling Modifiers
*   **Rule**: A modifying phrase must clearly modify the subject of the main clause.
*   **Real Number Demo**: Incorrect: *"Having eaten dinner, the computer was turned on."* $\implies$ Correct: *"Having eaten dinner, **I** turned on the computer."*

## 2. Active & Passive Voice Table

| Tense | Active Voice | Passive Voice |
| :--- | :--- | :--- |
| **Simple Present** | writes | is/are written |
| **Simple Past** | wrote | was/were written |
| **Present Continuous** | is writing | is/are being written |
| **Past Continuous** | was writing | was/were being written |
| **Present Perfect** | has written | has/have been written |
| **Future Simple** | will write | will be written |

---

## 3. Cloze Test Heuristics
*   **Collocations**: Look for words that naturally go together (e.g. "interested **in**", "keen **on**", "cope **with**").
*   **Tone Matching**: Check if the sentence tone is positive or negative, and select matching adjectives/adverbs.
*   **Transition Clues**: Look for logical transition words like "however", "consequently", "furthermore" to identify contrasting or supporting ideas.
