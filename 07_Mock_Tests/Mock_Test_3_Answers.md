## Section 1 — Numerical Ability Solutions

### Q1 — Number of Factors
*   **Correct Answer:** A (24)
*   **Derivation:**
    1.  Perform prime factorization of 360:
        *   $360 = 36 \times 10 = 6^2 \times 2 \times 5 = 2^3 \times 3^2 \times 5^1$.
    2.  Use the formula for number of factors: $F(N) = \prod (a_i + 1)$.
    3.  $F(360) = (3 + 1)(2 + 1)(1 + 1) = 4 \times 3 \times 2 = 24$.

### Q2 — Pipes & Cisterns
*   **Correct Answer:** B (30 hours)
*   **Derivation:**
    1.  Let total capacity of the tank be $\text{LCM}(12, 20) = 60$ units.
    2.  Efficiency of filling pipe = $60 / 12 = +5$ units/hour.
    3.  Efficiency of emptying pipe = $60 / 20 = -3$ units/hour.
    4.  Net efficiency when both are open = $5 - 3 = +2$ units/hour.
    5.  Time to fill the empty tank = $\text{Total Capacity} / \text{Net Efficiency} = 60 / 2 = 30$ hours.

### Q3 — Percentages & Commission
*   **Correct Answer:** B (32,500 Rupees)
*   **Derivation:**
    1.  Let total sales be $S$.
    2.  Commission on sales up to $10,000$: $10,000 \times 0.05 = 500$ Rupees.
    3.  Commission on sales exceeding $10,000$: $(S - 10,000) \times 0.04 = 0.04S - 400$ Rupees.
    4.  Total Commission = $500 + 0.04S - 400 = 0.04S + 100$ Rupees.
    5.  Amount remitted = $S - \text{Commission} = S - (0.04S + 100) = 0.96S - 100$ Rupees.
    6.  Given: $0.96S - 100 = 31,100 \rightarrow 0.96S = 31,200$.
    7.  $S = 31,200 / 0.96 = 32,500$ Rupees.

### Q4 — Simple & Compound Interest
*   **Correct Answer:** A (400 Rupees)
*   **Derivation:**
    1.  Let the principal sum be $P$.
    2.  Compound Interest for 2 years at $10\%$:
        *   $CI = P \left(1 + \frac{10}{100}\right)^2 - P = P(1.21 - 1) = 0.21P$.
    3.  Given $0.21P = 420 \rightarrow P = 420 / 0.21 = 2000$ Rupees.
    4.  Simple Interest on 2000 Rupees for 2 years at $10\%$:
        *   $SI = \frac{P \times R \times T}{100} = \frac{2000 \times 10 \times 2}{100} = 400$ Rupees.

### Q5 — Averages
*   **Correct Answer:** A (76 runs)
*   **Derivation:**
    1.  Total score in 10 innings = $10 \times 32 = 320$ runs.
    2.  New average desired after 11th inning = $32 + 4 = 36$ runs.
    3.  Total score in 11 innings = $11 \times 36 = 396$ runs.
    4.  Runs required in the 11th inning = $396 - 320 = 76$ runs.

### Q6 — Speed, Distance & Stream
*   **Correct Answer:** A (8 km)
*   **Derivation:**
    1.  Let the distance to the place be $D$ km.
    2.  Speed downstream = $\text{Speed of boat} + \text{Speed of current} = 6 + 2 = 8$ km/h.
    3.  Speed upstream = $\text{Speed of boat} - \text{Speed of current} = 6 - 2 = 4$ km/h.
    4.  Total time taken for the round trip is 3 hours:
        *   $\frac{D}{8} + \frac{D}{4} = 3 \rightarrow \frac{3D}{8} = 3 \rightarrow D = 8$ km.

### Q7 — Partnership Ratios
*   **Correct Answer:** A (10:12:15)
*   **Derivation:**
    1.  $\text{Profit Ratio} = \text{Capital Ratio} \times \text{Time Ratio}$.
    2.  $\text{Profit of A} : \text{Profit of B} : \text{Profit of C} = (2 \times 5) : (3 \times 4) : (5 \times 3) = 10 : 12 : 15$.

### Q8 — Probability
*   **Correct Answer:** A (1/9)
*   **Derivation:**
    1.  Total outcomes when two dice are thrown = $6 \times 6 = 36$.
    2.  Favorable outcomes for a sum of 9: $(3,6), (4,5), (5,4), (6,3) \rightarrow 4$ outcomes.
    3.  $\text{Probability} = 4 / 36 = 1 / 9$.

### Q9 — Data Interpretation
*   **Correct Answer:** A (66%)
*   **Derivation:**
    1.  Total students = 50.
    2.  Students scoring 20 or more marks belong to the ranges 20-30, 30-40, and 40-50.
    3.  Number of students scoring 20+ = $18 (20-30) + 10 (30-40) + 5 (40-50) = 33$ students.
    4.  $\text{Percentage} = (33 / 50) \times 100 = 66\%$.

### Q10 — Algebra (Indices)
*   **Correct Answer:** A (4)
*   **Derivation:**

## Section 4 — Coding Solution (Q31)

### Approach (Frequency Counting - O(N) Time, O(1) Space)
1.  If the lengths of strings `s` and `t` are not equal, they cannot be anagrams. Return `false`.
2.  Maintain a frequency count array of size 26 (for lowercase English characters).
3.  Iterate through string `s` and increment the count for each character, while iterating through string `t` and decrementing the count.
4.  If the strings are anagrams, all elements in the frequency array must be exactly 0.

### Optimized C++ Implementation
```cpp
#include <iostream>
#include <string>
#include <vector>

bool isAnagram(const std::string& s, const std::string& t) {
    if (s.length() != t.length()) {
        return false;
    }
    
    std::vector<int> charCount(26, 0);
    for (size_t i = 0; i < s.length(); ++i) {
        charCount[s[i] - 'a']++;
        charCount[t[i] - 'a']--;
    }
    
    for (int count : charCount) {
        if (count != 0) {
            return false;
        }
    }
    
    return true;
}

int main() {
    std::string s, t;
    if (!(std::cin >> s >> t)) return 0;
    
    if (isAnagram(s, t)) {
        std::cout << "true" << std::endl;
    } else {
        std::cout << "false" << std::endl;
    }
    
    return 0;
}
```
