## 🗺️ Grammar Trap Comparison Table

| Rule Category | Incorrect Usage | Correct Usage | Core Explanation |
| :--- | :--- | :--- | :--- |
| **Subject-Verb (Collective)** | The team *were* winning the project. | The team *was* winning the project. | Acting as a single unified unit takes a singular verb. |
| **Preposition Choice** | This system is different *than* that. | This system is different *from* that. | Always use "different from" in comparisons. |
| **Article Phonetics** | He is *a* SQL developer. | He is *an* SQL developer. | Vowel sounds (like "Es-Que-El") take "an". |

# PART A: Grammar & Vocabulary Notes

---

## 1. High-Frequency Vocabulary Categories

### Cause & Effect Words
* **Consequently**: As a result.
* **Thereby**: By that means (always followed by an "-ing" verb).
* **Precipitated**: Caused to happen suddenly.

* **Immediate Worked Example**: The critical server exploit **precipitated** an urgent security patch, **thereby** preventing a breach.
* **Shortcut Demo**: Spotting "-ing" after a blank indicates "thereby" is the target choice: *"...indices, [thereby] reducing latency."*

### Contrast & Concession Words
* **Albeit**: Although.
* **Notwithstanding**: In spite of.
* **Paradoxically**: Counter to expectations.

* **Immediate Worked Example**: The database migrations were successful, **albeit** extremely time-consuming.
* **Shortcut Demo**: If a sentence has positive and negative parts joined by a comma, use "albeit" to link them: *"[Positive], [albeit] [Negative]"*.

## 2. 10 Practice Sentence Completion Blanks

**Q1.** The database server crashed; ______ all active customer sessions were immediately terminated.
* **Hint**: Look for a word that means "as a logical consequence of this".
* **Solution**: **consequently** (or **hence**). This adverb links the cause (crash) to the effect (terminated sessions).

**Q2.** The lead architect optimized the SQL query indices, ______ reducing search latency by forty percent.
* **Hint**: The blank is followed by a verb ending in "-ing".
* **Solution**: **thereby**. In corporate contexts, "thereby" indicates the mechanism by which the result was achieved.

**Q3.** The unpatched legacy web server remains ______ to remote code execution exploits.
* **Hint**: Means "exposed to danger" or "weakly protected".
* **Solution**: **vulnerable**. Standard cybersecurity descriptor.

**Q4.** The project manager had to ______ the security guidelines to ensure compliance.
* **Hint**: Means "repeat for emphasis".
* **Solution**: **reiterate**.

**Q5.** The developer applied a temporary hotfix, ______ it was not a permanent solution.
* **Hint**: Means "although" or "even if it was".
* **Solution**: **albeit**. Used as a conjunction to introduce a concession.
