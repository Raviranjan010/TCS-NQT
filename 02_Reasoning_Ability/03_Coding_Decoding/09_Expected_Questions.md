# 03 Coding Decoding - Expected Questions

This file contains 15 TCS NQT-style expected questions based on 2022-2025 patterns.

**Q1.** If in a certain code language, 'NQT' is coded as 'PSV', how will 'TCS' be coded in that language?
*   **Type:** Letter Shifting Pattern (+2 shift)
*   **Answer:** N (+2) -> P, Q (+2) -> S, T (+2) -> V. Applying the same logic to TCS: T (+2) -> V, C (+2) -> E, S (+2) -> U. Code is VEU.
*   **Time target:** 30 seconds
*   **TCS frequency:** High (appears in 85% of papers)
